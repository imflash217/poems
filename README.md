# 1. नैहरवा 
```
---------------------------------------------
नैहरवा हमका ना भावे ..... नैहरवा ....
---------------------------------------------
साँई की नगरी परम अति सुंदर, जहां कोई जाय ना आवे .
चाँद सूरज जहां ना पानी, को संदेश सुनावे .
दरद यह साँई को सुनावे ..... नैहरवा ....
---------------------------------------------
आगे चलो पंथ नहीं सूझे, पीछे दोष लगावै .
क़ेहि विधि ससुरे जाऊँ मोरी सजनी, विराज ज़ोर जरावे .
विषय रस नाच नचावे ..... नैहरवा ....
---------------------------------------------
बिन सतगुरु अपनो नहीं कोई जो यह राह बतावै .
क़हत कबीर सुनो भाई साधो सपने नु पीतम आवे .
तपन यह जिया की बुझावे ..... नैहरवा ....
---------------------------------------------
~ संत कबीर 
```
